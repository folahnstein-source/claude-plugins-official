# Flugbuchungstool

Recherchiert und vergleicht buchbare Flugoptionen über mehrere Plattformen. Liefert eine strukturierte Vergleichstabelle mit Ranking, Preisdetails, Gepäckregeln und klaren Buchungsempfehlungen. Erkennt Preisfallen wie versteckte Gebühren und Self-Transfer-Risiken.

## Skills
- **flugbuchungstool** — Vollständiger Flugvergleich mit Buchungsempfehlung

## Usage
Auslöser: "Flug von Hamburg nach München", "günstigsten Flug finden", "Flugvergleich".
