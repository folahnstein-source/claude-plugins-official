# SPA Red-Flag Due Diligence (Buy-Side)

Screens and redlines Share Purchase Agreement drafts from the buyer's perspective. Works with both German and English SPAs. Generates a RED/AMBER/GREEN traffic-light report and inserts Word track-changes redlines directly into the seller's document.

## Skills
- **spa-red-flag-dd** — Full SPA screening, report generation, and redline markup

## Usage
Trigger by uploading an SPA draft or asking: "check this SPA", "red-flag this purchase agreement".

## Note
Reference SPA documents from specific deals are not included in this repository (confidential). The skill uses a methodology baseline file (`references/buy-side-baseline.md`) derived from prior Vorsprung transactions.
