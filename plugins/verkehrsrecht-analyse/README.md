# Verkehrsrecht-Analysetool

Analysiert Schreiben aus deutschen Verkehrsordnungswidrigkeiten. Führt vollständige Vorprüfung durch — Fristenkontrolle, formelle/materielle Prüfung, Rechtsprechungsrecherche — und erstellt sofort verwendbare Kanzleidokumente (Einspruch, Akteneinsicht, Fristwahrung).

## Skills
- **verkehrsrecht-analyse** — Vollständige Analyse und Schriftsatzentwürfe

## Usage
Auslöser: Bußgeldbescheid, Anhörungsbogen, OWi-Verfahren, Einspruch einlegen.
