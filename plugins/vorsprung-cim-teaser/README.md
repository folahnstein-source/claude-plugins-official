# Vorsprung CIM-to-Teaser Builder

Converts a seller CIM (Confidential Information Memorandum) into a first-draft investor teaser in Vorsprung Partners house style. Extracts investor-relevant facts, anonymises confidential identifiers, and produces a concise 2-page draft with a Redaction Log — ready for deal team review before external sharing.

## Skills
- **vorsprung-cim-teaser** — Full CIM extraction, anonymisation, and teaser generation

## Usage
Trigger by uploading a CIM and asking: "draft a teaser from this", "prepare investor materials", "anonymise this CIM".
