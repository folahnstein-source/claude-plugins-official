# Vorsprung Boilerplate — Verbatim Text Blocks

Use these blocks verbatim. Do not paraphrase, edit, or abbreviate.

---

## Confidentiality Header (every page, top or bottom)

```
STRICTLY CONFIDENTIAL AND PROPRIETARY | Not an offer, no investment advice
```

Alternative form (used in some teasers):
```
Strictly Private & Confidential
```

Use whichever form is consistent with any prior teaser on the same deal.
Default: `STRICTLY CONFIDENTIAL AND PROPRIETARY | Not an offer, no investment advice`

---

## Vorsprung Overview Paragraph

```
Vorsprung is a traditional deal-by-deal PE partnership and not a highly-levered LBO firm. The team has
extensive experience in working with German entrepreneurs and a proven capability of sourcing
deals outside the public M&A pipeline, having completed more than 40 transactions in the
Mittelstand.
```

Do not add a track record callout box (deployed capital, MoiC, or IRR figures) unless the deal team has explicitly confirmed current, approved figures for the specific teaser. When in doubt, omit entirely.

## Investment Summary Sidebar Icons (for PPTX production only)

When producing a PPTX, the Investment Summary sidebar uses four icon rows:

| Icon | Label |
|------|-------|
| Grid/sector icon | Sector: [sector name] |
| Globe icon | Geography: [country/region] |
| Euro/currency icon | Transaction Value: EUR Xm (EUR Xm equity) |
| Target/crosshair icon | Target Returns: >X.Xx MoiC, c.X% IRR |

In Markdown output, represent as a simple list:
```
**Sector:** [sector]
**Geography:** [country/region]
**Transaction Value:** EUR Xm (EUR Xm equity)
**Target Returns:** >X.Xx MoiC, c.X% IRR
```
