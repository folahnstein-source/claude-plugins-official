# Vorsprung Commercial Due Diligence

Runs a fast, investor-grade commercial DD on a German SME target. Produces a structured 10-section markdown report with an Attractive / Neutral / Not Attractive investment view, within the 48-hour Vorsprung speed standard.

## Skills
- **vorsprung-commercial-dd** — Generates the full DD report

## Usage
Trigger by asking: "quick DD on this company", "assess this target", or uploading a CIM.
