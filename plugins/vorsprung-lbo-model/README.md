# Vorsprung LBO Model

Build investment-grade LBO financial models for Vorsprung Partners' German SME succession deals. Follows the Vorsprung template methodology: sources & uses, debt schedule, integrated P&L / cash flow / balance sheet, and IRR/MoM returns output.

## Skills
- **vorsprung-lbo-model** — Builds the complete model as an `.xlsx` file

## Usage
Trigger by asking: "build me a model for this deal", "run the numbers on this target", or uploading a CIM/Factbook.
